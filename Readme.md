## Run Locally

Clone the project

```bash
  git clone https://github.com/kunjank31/playhub-backend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn or npm install
```

Start the server

```bash
  yarn dev or npm run dev
```
